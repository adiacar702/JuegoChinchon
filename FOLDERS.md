#

## Carpetas

### src

Contiene todo el código fuente de la aplicación.

### tests

Contiene las pruebas unitarias desarrolladas con JUnit.

### doc

Contiene la documentación JavaDoc generada a partir de la documentación indicada en el codigo fuente.

### markdowndoc

Contiene la documentación realizada en Visual Studio Code usando markdown.

### assets

Contiene imágenes realizadas para su utilización en el README u otros archivos para la documentación del proyecto.

## Paquetes de src

### src/ai

Contiene las clases e interfaz usadas para el comportamiento de cada IA.

### src/factory

Contiene la clase que usará el patron de diseño Factory.

### src/game

Contiene las clases que controlarán el transcurso de la partida.

### src/gameConfig

Contiene la clase que permitirá realizar la configuración de la partida usando el patron de diseño Singleton.

### src/playerUsers

Contiene las clases que definirán si un jugador es Humano o Maquina.

### src/resources

Contiene las clases que controlan el comportamiento de cada recurso.

### src/rules

Contiene las clases que indicarán el funcionamiento correcto de la partida utilizando metodos que harán comprobaciones en los turnos de los jugadores.

### src/utils

Contiene clases Extras que ayudarán a la correcta visualización de la partida en la terminal para que el juego sea mas intuitivo.
