# Herramientas Utilizadas en la creación del proyecto

## IDE Eclipse

- Realizar el codigo fuente del juego

## Documentación JavaDoc

- Crear la documentación sobre las distintas clases del juego.

## Junit 5

- Realizar diferentes pruebas de codigo sobre el proyecto.

## Draw.io

- Realizar la estructuración del juego con las distintas clases junto sus metodos y relaciones diferenciadas.

## GitHub

- plataforma en la nube para alojar y compartir el repositorio del proyecto.

## Git Bash

- aplicación de terminal para Windows para ejecutar comandos de Git usando texto.

## Visual Studio Code

- Usado junto con markdown para la documentación del proyecto.
